# Painter MFC project

Object Oriented Programming project in a visual studio environment using the MFC library.

The painter abilities are:

1) Save File.

2) Load File.

3) Undo / Redo.

4) Pick Shape.

5) Set Background.

6) Change Shape Color.

7) Change Pen Color.

8) Select Width Pen
